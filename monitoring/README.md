# Monitorización, auditoría y continuidad

Este directorio reserva el espacio técnico para documentación relacionada con:

- monitorización del entorno;
- logs y evidencias;
- SAI y pruebas de continuidad;
- scripts validados que deban versionarse.

## Estado

La arquitectura del proyecto contempla monitorización y auditoría, y la fase final incluye trabajo específico con el SAI. El volcado de evidencias exactas y del script definitivo debe hacerse únicamente desde la versión ya validada del proyecto, para mantener correspondencia total con la memoria final.
