# Plugins actuales — HUMANía Web

Este documento servirá para inventariar los plugins instalados actualmente en la web.

## Objetivo

Saber qué plugins existen, para qué sirven y si son necesarios en la futura estructura de HUMANía Web.

## Criterios de análisis

Cada plugin debe clasificarse según:

- función real
- importancia
- dependencia del contenido actual
- impacto en seguridad
- impacto en rendimiento
- alternativa propia
- alternativa externa mejor
- posibilidad de eliminación

## Categorías previstas

- SEO
- caché/rendimiento
- seguridad
- backups
- formularios
- newsletter/suscriptores
- cookies/privacidad
- analítica
- maquetación/diseño
- redes sociales
- podcast/audio
- automatización
- otros
