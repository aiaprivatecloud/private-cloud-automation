# Contenido actual — HUMANía Web

Este documento servirá para analizar el contenido existente en la web actual.

## Objetivo

Comprobar qué contenido debe conservarse y cómo debe adaptarse al nuevo diseño de HUMANía.

## Tipos de contenido a revisar

- entradas
- páginas
- imágenes
- categorías
- etiquetas
- menús
- formularios
- widgets
- shortcodes
- bloques reutilizables
- contenido del podcast
- noticias
- artículos editoriales

## Riesgos

- páginas construidas con maquetadores visuales
- shortcodes dependientes de plugins antiguos
- formularios que dejen de funcionar
- widgets obsoletos
- imágenes sin texto alternativo
- categorías duplicadas
- etiquetas desordenadas
- URLs que no deben cambiar

## Objetivo de migración

El nuevo tema debe poder mostrar el contenido actual sin perder:

- URLs importantes
- SEO
- imágenes destacadas
- categorías
- etiquetas
- estructura editorial
- enlaces internos
