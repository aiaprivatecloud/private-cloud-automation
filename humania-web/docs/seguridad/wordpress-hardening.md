# Hardening WordPress

Este documento recogerá las medidas de endurecimiento de WordPress para HUMANía Web.

## Medidas previstas

- 2FA en cuentas administradoras.
- Contraseñas robustas.
- Roles mínimos.
- SSL activo.
- Backups periódicos.
- Staging antes de producción.
- Plugins mínimos.
- Eliminación de plugins no usados.
- Actualizaciones controladas.
- Desactivar edición de archivos desde el panel.
- Revisar permisos de archivos.
- Configurar seguridad en Hostinger.
