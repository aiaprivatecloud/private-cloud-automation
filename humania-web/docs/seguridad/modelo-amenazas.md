# Modelo de amenazas — HUMANía Web

Este documento recogerá el modelo inicial de amenazas del proyecto.

## Activos a proteger

- Web pública.
- Cuenta de administración WordPress.
- Repositorio GitHub.
- Claves API.
- Sistema de automatización.
- Contenido editorial.
- Datos de formularios.
- Backups.

## Amenazas iniciales

- Acceso no autorizado.
- XSS.
- CSRF.
- Inyección.
- Prompt injection.
- Publicación automática incorrecta.
- Exposición de secretos.
- Manipulación de fuentes externas.
- Pérdida de datos.
- Dependencias vulnerables.
