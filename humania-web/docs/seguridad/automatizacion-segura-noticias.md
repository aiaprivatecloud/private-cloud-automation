# Automatización segura de noticias

Este documento recogerá las medidas de seguridad para el sistema de noticias automatizadas de HUMANía Web.

## Principios

- No publicar automáticamente al inicio.
- Crear borradores.
- Validar fuentes.
- Eliminar duplicados.
- Validar URLs.
- Controlar errores.
- Registrar eventos.
- Revisar contenido generado por IA.
- Prevenir prompt injection.
- No exponer claves API.

## Pendiente

- Lista de fuentes permitidas.
- Flujo de validación.
- Flujo de revisión humana.
- Política de publicación.
- Sistema de logs.
