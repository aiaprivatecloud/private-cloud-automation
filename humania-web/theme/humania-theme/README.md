# HUMANia Theme

Tema propio de HUMANia Web.

## Objetivo

Crear un tema WordPress ligero, accesible, mobile-first y preparado para contenido editorial sobre inteligencia artificial.

## Principios

- Mobile-first.
- Accesibilidad.
- Seguridad desde el diseño.
- Rendimiento.
- HTML semántico.
- CSS claro y mantenible.
- JavaScript mínimo.
- Sin dependencias innecesarias.

## Archivos iniciales

- style.css
- functions.php
- index.php
- header.php
- footer.php
- assets/css/main.css
- assets/js/main.js
