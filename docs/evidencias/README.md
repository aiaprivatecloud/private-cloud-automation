# Evidencias técnicas

Este directorio se reserva para evidencias anonimizadas y útiles para defensa técnica.

## Subcarpetas

- `capturas/`: imágenes técnicas o capturas relevantes.
- `logs/`: registros seleccionados y no sensibles.

## Regla básica

No subir:

- contraseñas;
- claves privadas;
- direcciones públicas no necesarias;
- copias de seguridad reales;
- datos personales;
- exportaciones completas de configuraciones sensibles sin anonimizar.
